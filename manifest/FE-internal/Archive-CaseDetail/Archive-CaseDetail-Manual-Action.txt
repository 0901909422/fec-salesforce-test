1) Open the Setup page >> External Objects >> Choose "AthenaCases"
>> Buttons, Links, and Actions >> Click Edit "View"
In "Salesforce Classic Override" choose Visualforce page.
In Combobox, choose "ArchiveCaseRedirect". Then click Save


2) Open the Developer Console and run this script to insert the Schedule Job

String sched = '0 0 0 * * ?'; // Daily Midnight 0:00
DeleteArchiveCaseBatch delArchiveCase = new DeleteArchiveCaseBatch();
System.schedule('Delete Archive Case Temp', sched, delArchiveCase);
